# 比較運算子

1. \=\= (相等)
    
    ```php
    //It true
    //當比較值其中一為 int 時，則將字串轉為 int
    $a = 3;
    $b = "3b";
    //php 8.0 修正
    if($a == $b) {
    	echo "It true";
    }
    ```
    
2. \=\=\= (完全相等)
3. \!\= or \<\> (不等於)
4. \!\=\= (完全不等於)
5. \> (大於)
6. \>= (大於等於)
7. \< (小於)
8. \<\= (小於等於)
9. \<\=\> (太空船)  
    1. 左 > 右 時 = 1  
    2. 左 < 右 時 = -1  
    3. 左 == 右 時 = 0
    ```php
    //$c = 1
    $a = 5;
    $b = 0;
    $c = $a <=> $b;
    
    //$c = 0
    $a = 10;
    $b = 10;
    $c = $a <=> $b;
    
    //$c = -1
    $a = 9;
    $b = 15;
    $c = $a <=> $b;
    ```
